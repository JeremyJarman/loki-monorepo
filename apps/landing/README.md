# LOKI Landing Page

> A modern, professional landing page for **LOKI** - the world's first Experience Sharing network. Discover local venues, events, and specials. Share with friends. Make plans together.

## About LOKI

LOKI is the world's first Experience Sharing network. Just as Instagram is for images and Spotify is for music, LOKI is for where you're going next. The platform answers the question: **"What's happening around me tonight?"** and enables users to discover venues, share plans with friends, and make social decisions together.

## What This Site Does

This landing page serves as the digital front door for LOKI, designed to:

- **Drive app downloads** and capture investor interest
- **Build an email waitlist** for early access users
- **Establish brand credibility** with a modern, professional design
- **Communicate the value proposition** clearly and compellingly

## Site Sections

### Hero Section
The first thing visitors see - a bold headline "Find everything." that immediately communicates LOKI's purpose. Features a clear value proposition and call-to-action buttons to download the app or join the waitlist.

### Problem Section
Addresses the "What Now?" dilemma that users face when trying to discover local events and venues. Highlights three key pain points:
- **Algorithm Burying** - Social media algorithms hide important posts
- **Fragmented Information** - Details scattered across multiple platforms
- **No Central Hub** - No single place to discover and share plans with friends

### Solution Section
Showcases LOKI's unique features and how it solves the discovery problem. Highlights the platform's key differentiators and benefits.

### How It Works
A step-by-step guide showing users exactly how LOKI works - from discovery to sharing with friends. Makes the platform approachable and easy to understand.

### Business Value
Demonstrates the value LOKI brings to venues and partners, showing why businesses should join the platform.

### Call-to-Action
A compelling section encouraging visitors to join the waitlist or contact the team. Features email capture forms and contact information.

### Footer
Complete site navigation, social media links, and company information for easy access.

##Key Features

- **Mobile-First Design** - Optimized for the devices where LOKI will be used most
- **Smooth Animations** - Engaging transitions that guide the user's eye
- **Email Capture** - Easy-to-use forms for building the waitlist
- **SEO Optimized** - Built to rank well in search engines
- **Fast Loading** - Optimized for speed and performance
- **Accessible** - Designed to be usable by everyone

## Design Philosophy

The landing page uses a carefully crafted design system:

- **Colors**: Deep forest green (#122220) as the primary brand color, vibrant orange (#FF3D00) for calls-to-action, and olive green (#485C11) as an accent
- **Typography**: Elegant serif font (Crimson Text) for headlines, clean sans-serif (DM Sans) for body text
- **Layout**: Clean, spacious design that puts content first
- **Tone**: Professional yet approachable, modern yet timeless

## Quick Start

If you're a developer looking to work on this site:

1. **Install dependencies**: `npm install`
2. **Start development server**: `npm run dev`
3. **Build for production**: `npm run build`

The site is built with React, TypeScript, TailwindCSS, and Vite for modern, fast development and deployment.
